# Hermes TodoList

> Production-grade todo list application — built with full SDLC, powered by Hermes Agent.

## 🎯 Vision

A simple, fast, team-focused todo list application (MVP for 5–20 users), designed to scale into a public SaaS platform.

## 🏗️ Tech Stack

| Layer | Technology |
|-------|-----------|
| **Frontend (Web)** | Next.js |
| **Mobile** | Flutter (backlog) |
| **Desktop** | Electron (backlog) |
| **Backend API** | Golang |
| **Database** | PostgreSQL |
| **Deployment** | Docker + VPS |
| **CI/CD** | GitHub Actions |
| **UX/UI** | Figma |

## 📋 MVP Features

- [x] User authentication (username + password)
- [x] Task CRUD: title, description, due date, priority, tags, assignee, status
- [x] Simple list view
